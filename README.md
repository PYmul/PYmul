# PYmul --- Universal Python Manager

PYmul it's a library that makes Python commands identical across any operating system.
The idea is that it doesn't matter if you're on **Windows, Linux, Android, macOS, or iOS**, the command works the same.

Example:

bash
rel install flask

It works on any system, even though the original command on Linux is "apt install", and on Windows it's "pip install".
----------------------------------------------------------------------------------------------------------------------
## Objective

To create a "universal manager" for:

- installing packages
- creating environments
- launching programs
- pulling system information
- abstracting OS differences
